# Installation
## Prerequisites

- Node.js version = 22.x
- npm version >= 10.9.2

A `.tool-versions` file is provided to easily work with [asdf](https://asdf-vm.com/)


1. Clone the repository:
  ```bash
  git clone https://github.com/vinjatovix/aft-actors-script.git
  cd aft-actors-script
  ```

2. Install dependencies:
  ```bash
  npm install
  ```

  ## Usage
  Please rename `.env_example` to `.env` And full-fill the variables if needed.

  You can install the current local Back end following [these instructions](https://github.com/vinjatovix/ts-api)

  ### Run in dev mode
Run the development server:
```bash
npm run dev
```
